# Library Management System (SQLite)

## 📌 Overview

This project is a Library Management System built using Java with SQLite for data persistence. It allows users to manage books and user accounts with stored data that remains even after closing the application.

## ✨ Features

* Add, update, and delete books
* User account registration and login
* Persistent data storage using SQLite
* Book recommendations (optional)
* External Book API integration (optional)

## 🛠️ Technologies Used

* Java
* SQLite
* JavaFX / Swing (for GUI)

## 📚 Key Concepts

* Data Persistence with SQLite
* Database Integration
* User Management
* CRUD Operations
* API Integration (optional)

## 🚀 How to Run

1. Install Java (JDK 8 or above)
2. Install SQLite (or use built-in driver)
3. Clone or download this repository
4. Open in IDE (IntelliJ / Eclipse)
5. Run the main class

## 📂 Project Structure

* Main Class (application entry point)
* Database Layer (SQLite connection & queries)
* UI Components (forms, buttons, tables)
* Logic Layer (book & user operations)

## 🎯 Purpose

This project helps in understanding how real-world systems store and manage data persistently using databases.
